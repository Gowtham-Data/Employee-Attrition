# Employee-Attrition
Employee Attrition Analysis and Prediction

This project predicts whether an employee will **Stay** or **Leave** a company using a Machine Learning model.

--

## Objective
To help HR teams identify at-risk employees early and take retention actions.

---

# Input Features
The prediction is made using only 6 important fields:

- Age  
- Monthly Income  
- Years at Company  
- OverTime (Yes/No)  
- Job Role  
- Marital Status  

---

##Model Used
Random Forest Classifier
